Build
-------------

    make lib

The binary library is to be found in the lib directory.

Unit Test
--------------

Get Unity from here: http://www.throwtheswitch.org/unity/. Copy all the files to the Unity-master subdirectory. Then, run:

    make utest
    utest/utest
